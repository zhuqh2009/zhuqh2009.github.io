### Maven 常用命令记录

#### 1、手动把jar包安装到本地Maven仓库

```
//maven 手动把本地jar安装到本地仓库

mvn install:install-file  -Dfile=D:/jar/xxx.jar  -DgroupId=xxx.xxx  -DartifactId=xxx -Dversion=x.x -Dpackaging=jar

```

#### 2、 跳过Test跳过gpg打包

```
mvn clean install -Dmaven.test.skip=true -Dgpg.skip
```
